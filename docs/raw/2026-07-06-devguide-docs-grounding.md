# 2026-07-06 — dev-guide 근거 범위 번복 지시 (코드만 → 코드 + 레포 문서, 코딩 규칙 포함)

> 출처: wiki_pipeline 세션 논의. 이 파일은 불변 원본이다.

## 맥락

같은 날 grilling([[2026-07-06-theme-detail-grilling]]) Q2에서 dev-guide 범위를 "1문서 · 코드 근거만
(조직 규칙 제외)"으로 확정했고, 직후 확인 문답("컨벤션 포함으로 수정할까요?")에서도 "현재 결정 유지"를
선택했다. 그 직후 사용자가 아래와 같이 번복 지시함.

## 사용자 지시 (원문)

"아니야 dev-guide는 코드 뿐만 아니라 문서를 탐색해서 코딩 규칙도 찾아서 정리해야해"

## 해석 (세션에서 명시한 전제)

- 탐색 대상 문서 = **레포 내 문서** (README·CONTRIBUTING·docs/ 등 러너가 clone으로 받는 범위)
- 레포 밖 외부 시스템(사내 위키 등) 탐색은 엔진 접근 범위 밖 — 필요 시 별도 결정
