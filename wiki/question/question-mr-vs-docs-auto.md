# ❓ 산출물 저장 — MR 방식으로 최종 확정할 것인가?

원 설계(docs-auto 브랜치 + 중앙 배치 pull)를 docs-hub 직접 MR로 바꾸는 제안([[decision-mr-review-gate]])이
팀 확정을 받았다. MR 정책 세부(리뷰어 자동 지정 규칙, 소스별 MR vs 하루치 통합 MR)는 [[decision-mr-review-gate]]의 MR 규격을 따른다.

## ✅ 답 (2026-07-05)

**docs-hub 직접 MR 방식으로 최종 확정** → [[decision-mr-review-gate]] (확정 상태로 갱신됨). 소스별 1 MR + 사람 리뷰 게이트.
