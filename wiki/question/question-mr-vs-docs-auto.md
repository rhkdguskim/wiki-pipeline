# ❓ 산출물 저장 — MR 방식으로 최종 확정할 것인가?

원 설계(docs-auto 브랜치 + 중앙 배치 pull)를 docs-hub 직접 MR로 바꾸는 제안([[decision-mr-review-gate]])이
아직 팀 확정을 받지 않았다. MR 정책 세부(리뷰어 자동 지정 규칙, 소스별 MR vs 하루치 통합 MR)도 함께 정해야 한다.

- **블로킹**: Phase 1의 제출 단계(MR/PR 생성) 구현 방향을 가름
- 확정되면: [[decision-mr-review-gate]]를 확정 상태로 갱신하고 이 질문을 `answered`로 전환
