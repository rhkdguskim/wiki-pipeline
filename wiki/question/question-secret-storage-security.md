# ❓ 등록 시크릿의 저장 보안은 어떻게 보장하나?

> **우선순위 낮음 (2026-07-05 사용자 판단)** — 지금 단계에선 중요하지 않다. 실제 운영·배포에 가까워질 때 다룬다. 설계 진행의 게이트가 아님.
